4. Analisis Kompleksitas:
- Membuat Input Data array :  O(n)
  penjelasan: karena untuk mengalokasikan memori dan elemen memerlukan waktu yang sebanding dengan jumlah elemen.
- Mencari Max/Min dan Rata-rata:  O(n)
  Penjelasan: karena program yang telah saya buat fungsinya harus memeriksa atau menjumlahkan setiap elemen dalam array  untuk menentukan min, max, dan rata-rata.
- Menentukan Data Kelulusan:  O(n)
  Penjelasan: karena program ini menggunakan list comprehension untuk mengecek setiap nilai, setiap data akan dicek sehinggan dapat dipisah antara lulus dan tidak lulus.
- Visualisasi:  O(n)
  Penjelasan: fungsi ini digunakan untuk memproses data menjadi grafik.
- Kesimpulan: Secara keseluruhan, program yang telah saya buat ini berjalan dalam waktu linear atau  O(n) .
